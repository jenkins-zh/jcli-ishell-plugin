# jcli-ishell-plugin
This is a plugin for [Jenkins CLI](https://github.com/jenkins-zh/jenkins-cli/) which allows you to manage your jobs in an interactive way.

# Get started
```
jcli config plugin fetch
jcli config plugin install ishell
```
